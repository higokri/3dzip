# 3DZip — Project Page

Project page for **3DZip: Spatial-Aware Feature Diversity-Guided Token Compression for 3D Question Answering** (ECCV 2026).

Changwoo Baek, Kyeongbo Kong — Pusan National University.

## Local preview

```bash
python3 -m http.server 8000
# open http://localhost:8000
```

## Deploy with GitHub Pages

1. Push this repository to GitHub.
2. In **Settings → Pages**, set the source to the `main` branch, `/ (root)` folder.
3. The page will be served at `https://<user>.github.io/<repo>/`.
